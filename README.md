# 📊 Customer Churn Analysis Project

## 📌 Project Overview
This project focuses on analyzing customer churn behavior to understand why customers leave a service and identify key patterns that can help improve customer retention.

The analysis is done using Python and data visualization libraries.

---

## 🎯 Objective
- Analyze customer churn patterns
- Identify factors affecting churn
- Understand customer behavior based on tenure, charges, and demographics
- Provide business insights to reduce churn

---

## 📂 Dataset Information
The dataset contains customer details such as:
- Gender
- Tenure
- Monthly Charges
- Total Charges
- Churn status (Yes/No)

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Key Analysis Performed

### 1. Overall Churn Distribution
Visualized how many customers stayed vs left.

### 2. Gender vs Churn
Analyzed if gender impacts churn behavior.

### 3. Monthly Charges vs Churn
Checked if higher charges lead to higher churn.

### 4. Tenure vs Churn
Studied relationship between customer tenure and churn.

---

## 📈 Key Insights
- Customers with lower tenure are more likely to churn.
- Higher monthly charges are associated with higher churn.
- Gender does not significantly affect churn behavior.
- Retaining long-term customers is easier compared to new customers.

---

## 🚀 Conclusion
This analysis helps businesses understand customer behavior and take actions to reduce churn, improve customer satisfaction, and increase retention.
